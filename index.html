Módulo 1: Diseño de Sistemas Informáticos

Escribir código sin un plano previo es como intentar construir un edificio sin planos arquitectónicos: el colapso está casi asegurado. Este módulo está diseñado para enseñarte a pensar como un arquitecto de software. Aprenderás a traducir requerimientos de negocio complejos en modelos visuales estandarizados utilizando UML (Unified Modeling Language). A lo largo del curso, descubrirás cómo documentar, estructurar y comunicar la arquitectura de una aplicación antes de que se escriba la primera línea de código. Este módulo cierra la brecha entre las necesidades del cliente y el equipo de desarrollo, asegurando que los sistemas creados sean escalables, robustos y mantenibles.

Módulo 2: Programación Orientada a Objetos

Este módulo está diseñado para transformar la forma en que piensas, estructuras y escribes código. Dejaremos atrás la programación secuencial para sumergirnos de lleno en el paradigma de Programación Orientada a Objetos (POO) utilizando Java, uno de los lenguajes más robustos, escalables y demandados en la industria del software. A lo largo del módulo, aprenderás a modelar problemas del mundo real mediante abstracciones digitales. Dejarás de escribir código suelto para empezar a diseñar sistemas de software reales mediante componentes independientes, reutilizables y fáciles de mantener.

Módulo 3: Diseño de Arquitectura de Software

Te damos la bienvenida al módulo de Diseño de Arquitectura de Software! En el ecosistema tecnológico actual, las aplicaciones no escalan ni sobreviven por accidente; lo hacen gracias a una base estructural sólida. Este módulo está diseñado para transformarte de un programador que solo escribe código a un arquitecto que diseña soluciones robustas, eficientes y mantenibles. Durante este bloque, nos sumergiremos de lleno en el desarrollo de APIs RESTful, el estándar de oro para la comunicación en la web moderna. Utilizaremos Spring Boot, el framework líder en la industria para el ecosistema Java, conocido por su capacidad de simplificar la configuración y acelerar el despliegue de aplicaciones de nivel empresarial.

Módulo 4: Programación de Componentes de Bases de Datos

Una base de datos no es solo un almacén pasivo de información; es el motor inteligente donde reside el activo más valioso de cualquier organización: sus datos. En este módulo, dejarás atrás las consultas básicas de manipulación de datos para convertirte en un desarrollador de bases de datos backend. Utilizando Oracle Database, el estándar de oro para entornos empresariales y sistemas de alta transaccionalidad, aprenderás a programar lógica de negocio directamente en el servidor mediante PL/SQL (Procedural Language/Structured Query Language). Al centralizar la lógica en la base de datos, lograrás que los sistemas creados en los módulos anteriores (como tu aplicación en Java) sean increíblemente rápidos, seguros y eficientes.
