Clase 1: Fundamentos de Diseño de Sistemas y modelos SDLC

El Diseño de Sistemas es el proceso de definir la arquitectura, los componentes, los módulos, las interfaces y los datos de un sistema para satisfacer requisitos específicos. No se trata de qué lenguaje de programación usar, sino de cómo las piezas del rompecabezas encajan para formar una imagen coherente, robusta y eficiente.
